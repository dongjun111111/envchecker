# envchecker
环境配置检测工具

-  默认开放端口: 8080
-  配置文件: config.ini
-  启动: go run *.go
-  支持组件：
    - APM
    - Apollo
    - Consul
    - Elasticsearch
    - Kafka
    - MySQL
    - Redis
    - Syslog
    - Clickhouse
    - PostgreSQL
