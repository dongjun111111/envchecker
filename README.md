# envchecker
环境配置检测工具

- 应用场景：
    - 应用迁移前全量配置可用性检查
    - 项目定期账号密码更新前可用性检查
 

-  默认开放端口: 8080
-  配置文件: config.ini
-  启动: go run *.go
-  支持组件：
    - APM
    - Apollo
    - Consul
    - Elasticsearch
    - Kafka
    - MySQL
    - Redis
    - Syslog
    - Clickhouse
    - PostgreSQL
    - Flink
    - Hbase
    - Tidb
