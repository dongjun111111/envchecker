# envchecker
环境配置检测工具

- 注意：
    - 如果需要部署多处，则需要将HTML静态文件打包进可执行文件中。可使用 [packr](https://github.com/gobuffalo/packr) 工具
-  默认开放端口: 8080
-  配置文件: config.ini
-  启动: go run *.go
-  支持组件：
    - APM
    - Apollo
    - Consul
    - Elasticsearch
    - Kafka
    - MySQL
    - Redis
    - Syslog
    - Clickhouse
    - PostgreSQL
    - Flink
    - Hbase
    - Tidb
